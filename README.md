# eslint config

## Install

### JavaScript

```sh
npm install --save-dev eslint @cabe/eslint-config
```

```sh
npm install --save-dev eslint @cabe/eslint-config @cabe/eslint-config-typescript @typescript-eslint/eslint-plugin @typescript-eslint/parser
```

## .eslintrc

```json
{
  "extends": [
    "@cabe",
    "@cabe/typescript"
   ]
}
```
